This is a full-stack e-wallet application designed to provide users with a simple, fast, and secure way to manage digital transactions. Users can create accounts, send and receive money, track their transaction history, and monitor their balance in real time.

The project focuses on building a strong foundation in authentication, backend development, and modern UI design using tools like JWT for secure login, and a responsive frontend for a smooth user experience.

It is built as a learning and portfolio project to practice real-world development concepts such as user authentication, API design, state management, and secure financial workflows.

✨ Key Features:

User authentication (JWT-based login system)
Secure money transfer system
Real-time balance tracking
Transaction history logs
Responsive and clean UI design
Basic “Remember Me” login feature
Google OAuth (planned or optional)
